## 打包提交流程

打开终端，输入命令进入`project`目录， 然后对model文件进行压缩，最后使用命令进行提交，将token替换成自己的即可。

```
cd  /home/workspace/project

zip -r models.zip model

castlecli --third sany --source /home/workspace/project/models.zip --token *********
```
